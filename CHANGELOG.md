## 1.0.0

### 🚀 Stable Version Released!

Initial release of `lacus.utils` on PyPI: reusable utility functions for Lacus Solutions' Python packages. Main features:

- **Type description**: `describe_type` returns Python-native type labels (`NoneType`, `dict`, `tuple`, built-ins, and list notation)
- **Random sequences**: `generate_random_sequence` with `SequenceType` for numeric, alphabetic, or alphanumeric output
- **Zero dependencies**: no external runtime packages required

For detailed usage and API reference, see the [README](./README.md).
